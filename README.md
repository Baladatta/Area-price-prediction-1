# 🏠 House Price Predictor

This is a simple machine learning project that predicts house prices based on area using Linear Regression. It includes:

- 🔍 Model training using `scikit-learn`
- 🌐 Web application built with Flask
- 📦 Deployment-ready setup using Render (includes `Procfile` and `gunicorn`)

---

## 🚀 Demo

Enter an area in square feet and get the estimated house price instantly.
